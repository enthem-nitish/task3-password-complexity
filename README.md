## рҹӣЎпёҸ Security Criteria Breakdown

<div align="center">

| Criteria | Requirement | Why It Matters |
|:---:|:---:|:---|
| рҹ“Ҹ **Length** | вүҘ 8 characters | Longer = exponentially harder to brute-force |
| рҹ”  **Uppercase** | At least one `A-Z` | Adds unpredictability to the character set |
| рҹ”Ў **Lowercase** | At least one `a-z` | Baseline for structural complexity |
| рҹ”ў **Numbers** | At least one `0-9` | Defeats simple dictionary attacks |
| вңЁ **Special Chars** | At least one `!@#$%^&*` | Maximum entropy against all attack vectors |
